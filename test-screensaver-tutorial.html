🔧 Integration Options

Option 1: Standalone Preview

Simply open screensaver-tutorial.html in your browser. Perfect for testing and previewing.

Option 2: Replace Video Screensaver

Edit js/reset.js to open this tutorial instead of playing videos. Useful if you want fully interactive idle experience.

Option 3: Hybrid Mode (Recommended)

Alternate between video screensaver and interactive tutorial. Show tutorial every 5th idle trigger. Keeps experience fresh and prevents user fatigue.

Integration Code

// In js/reset.js, replace startScreensaver() function: function startScreensaver() { inScreensaverMode = true; const tutorialWindow = window.open( 'screensaver-tutorial.html', 'screensaver', 'fullscreen=yes,menubar=no,toolbar=no' ); if (tutorialWindow) { tutorialWindow.focus(); window.closeScreensaver = function() { tutorialWindow.close(); stopScreensaver(); }; } }

Modify Animations

  • Slide timing: Adjust animation-delay values

❓ FAQ

Can I embed this in an iframe?

Yes, but fullscreen mode won't work. For best experience, use window.open() for fullscreen launch.

How do I customize the slides?

Edit HTML content in each .carousel-section div. Update text, emoji, and styling as needed.

Does it work on mobile?

Yes! Responsive design works on phones. Touch to exit, arrow indicators appear on small screens.

Can I add more slides?

Yes. Duplicate a .carousel-section div, increment the id number, and update TOTAL_SLIDES variable.

What if it doesn't display correctly?

Check browser console (F12), verify backdrop-filter is supported, ensure CSS is loaded correctly.
